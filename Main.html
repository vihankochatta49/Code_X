DOM stands for Document Object Model.

When an HTML document is loaded in web browser, it becomes a document object.

It is the data representation of the objects that comprise the structure and content of a document on the web.

It is an object-oriented representation of the web page, it can be modified with a scripting language such as JavaScript.

Explanation

The addEventListener() method binds an event handler to the document.

This method binds an event handler to the whole document, to bind event handler to a specified element use element.addEventListener method.

Use removeEventListener method to remove the event handler that has been attached with the addEventListener method.

Syntax

document.addEventListener(event, function)

Parameters discription

event: Mandatory. A word that specifies the name of the event.
IMP: Don't use the on prefix like use 'mousedown' instead of 'onmousedown'.

function: Mandatory. Specifies the function to run when event occur.

The anchors collection returns a collection of all elements in the document that have a name attribute.

The elements in the collection are sorted as they appear in the source code.

Syntax

document.anchors

How to access?

length: Returns the total number of <a> element with name attribute in the document.

[index] OR item(index): Returns the <a> element from the collection with secified index (starts at 0).

namedItem('id') Returns the <a> element from the collection with the specified id.

The original node is removed from the other document.

Syntax

document.adoptNode(node)

Explanation

The getElementById() method returns the element that has the ID attribute with the specified value.

This is the most used method in the HTML DOM, when you want to get or change info of an element.

If the element with specified ID does not exist it returns null.

An ID should be unique within a page. However, if more than one element with the specified ID exists, this method returns the first element in the source code.

Syntax

document.getElementById('elementID')

Explanation

The getElementsByClassName() method returns a collection of all elements in the document with the specified class name, as an array-like object.

The array-like object represents a collection of nodes. The nodes can be accessed by index numbers. The index starts at 0.

Elements are sorted in array-like object as they are in the document.

Syntax

document.getElementsByClassName('classname')

The querySelector() method returns the first element that matches a specified selector(s) in the document.

This method only returns the first element that matches the specified selectors.

If the selector matches an class in the document that is used several times, it returns the first matching element.

Syntax

document.querySelector('selector')

Explanation

The querySelectorAll() method returns all elements in the document that matches a specified selector(s), as a static NodeList object.

The NodeList object represents a collection of nodes. The nodes can be accessed by index numbers. The index starts at 0.

Syntax
